Title:
Elementary simulation of tethered Brownian motion

DEPOSIT INFORMATION

Description:
Two files are deposited:
1) "TPMdata_x_20ms.dat" is a text file of only the x values for a ~480 nm bead tethered to a 3477 basepair strand of DNA.  The data is acquired every 20 ms.
2) "Tethered Particle Analysis.m" is the MATLAB m-file that performs the analysis discussed in the text.
